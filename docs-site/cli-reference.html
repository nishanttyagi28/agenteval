CLI Reference

Every AgentEval subcommand accepts --help for the authoritative, current flag list. This page summarizes what each one does.

agenteval run

Run a registered agent's golden suite, score it, and save a JSON report.

agenteval run --agent my_agent
agenteval run --all
agenteval run --agent my_agent --case-id total_customers --tag smoke
agenteval run --agent my_agent --repeat 5 --repeat-case total_customers

agenteval compare

Compare a current run with a versioned baseline and apply the regression gate.

agenteval compare --agent my_agent
agenteval compare --agent my_agent --max-cost-increase-pct 20 --max-latency-p95-ms 3000
agenteval compare --agent my_agent --require-statistical-significance --significance-alpha 0.05

agenteval report

Generate a self-contained HTML report for a run, with baseline deltas and trend history.

agenteval report --agent my_agent

agenteval init

Scaffold agents.yaml, a sample golden suite, and a CI workflow for a new project.

agenteval init --agent-name my_agent --framework auto

agenteval generate

Generate reviewable adversarial variants of existing golden cases.

agenteval generate --cases tests/golden/my_cases.yaml --variants 3

agenteval import

Convert an external CSV dataset into golden test cases via a column-mapping config.

agenteval import --emit-mapping-template mapping.yaml
agenteval import data.csv --mapping mapping.yaml --output tests/golden/imported.yaml

agenteval generate-cases

Propose candidate golden cases from production run logs, or targeted regression cases from a baseline/current failure transition.

agenteval generate-cases --logs runs/<run>.json
agenteval generate-cases --logs sample_logs.jsonl --format jsonl
agenteval generate-cases --from-failures --baseline runs/baseline.json --current runs/latest.json

agenteval compare-models

Run the same golden suite against multiple registered agents and compare their metrics.

agenteval compare-models --agent my_agent_v1 --agent my_agent_v2

agenteval trace

Replay one case's step-by-step execution trace from a saved run, as text or a self-contained HTML page.

agenteval trace runs/<run>.json --case-id total_customers
agenteval trace runs/<run>.json --case-id total_customers --html trace.html

agenteval calibrate

Score LLM-judge/human agreement (Cohen's kappa) against a labeled calibration set.

agenteval calibrate --judge agentic_data_analyst --golden-set tests/golden/calibration_example.yaml

agenteval serve

Run a local, read-only dashboard-data API (no auth, localhost only -- see the README's Deployment section).

agenteval serve --local --agent agentic_data_analyst --port 8765

agenteval audit-log

Query the opt-in structured audit trail for one agent.

agenteval audit-log --agent agentic_data_analyst --since 2026-07-01
Flags and defaults change as AgentEval evolves — agenteval <command> --help is always the source of truth.
